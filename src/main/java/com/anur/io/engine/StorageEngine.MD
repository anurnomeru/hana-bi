##### 1、带 MVCC 的版本有何设计上的区别？

（1）我们将内存段拆分为未提交段 unCommit 与已经提交段 committed。

（2）我们允许同一段中有同样的key，但是版本不同。

有何好处？

我们可以无需过多考虑隔离性的问题，也可以在频繁增删改同一值时，进行快照的保存。
